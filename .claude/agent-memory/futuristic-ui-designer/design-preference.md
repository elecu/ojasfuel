**Preference:** Outlined buttons (borders, no fill) + flat artistic aesthetic.

**Why:** More sophisticated, modern, Bauhaus-inspired look. Less busy than filled buttons. References shown: clean bordered rectangles with text, flat design with bold colors (orange, red on black).

**How to apply:** 
- Primary buttons: border-only style, no background fill (or very transparent)
- Text centered, generous padding
- Use accent color for border, darker text
- Maintain green accent but apply to borders/outlines only
- Increase whitespace between elements
- Typography: clean, confident, minimal
- Header: Logo centered with balanced columns [1, 2, 1], not asymmetric
- Logo background must be transparent (CSS: background-color: transparent)
- Settings button aligned right for balance
